# CatalogoApp

🎮 Catálogo Gamer - ASP.NET MVC

Aplicación web desarrollada en ASP.NET Core MVC con arquitectura por capas, enfocada en la administración y exploración de un catálogo de videojuegos.

El proyecto permite visualizar videojuegos, agregar nuevos títulos, registrarse, iniciar sesión y realizar calificaciones con comentarios utilizando un sistema persistente basado en archivos JSON.

🚀 Características
🔐 Inicio de sesión y registro de usuarios
🎮 Catálogo de videojuegos
➕ Agregar nuevos videojuegos
⭐ Sistema de calificaciones de 0 a 5 estrellas
💬 Comentarios y reseñas
💾 Persistencia de reseñas en archivos .json
🎨 Interfaz Cyberpunk / Neon Gamer
✨ Animaciones y efectos visuales personalizados
📂 Arquitectura por capas
